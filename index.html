About Me

Hi! I’m a PhD student at Georgia Tech working with Prof. Moin Qureshi in the Memory Systems Lab. I leverage computer architecture concepts to tackle fun problems. Before joining Tech, I completed my undergrad from IIT Kanpur.

My recent works include practical and secure defenses against the severe Rowhammer security threat, and bandwidth-centric server designs using the emerging CXL memory interconnect.

I am currently exploring system-level optimizations to improve Large Language Model training and serving, a problem I first examined as part of my internship at AMD Research in Summer’23.
