About Me

Hi! I’m a PhD student at Georgia Tech working with Prof. Moin Qureshi in the Future Architecture and Systems Lab. I leverage computer architecture concepts to tackle fun problems. Before joining Tech, I was an undergrad at IIT Kanpur.

Currently, I am working on data-movement optimizations across the large language model serving stack—LLM arch, system software, and memory systems—to provide step-function improvements in LLM serving efficiency. In support of this work, I have been awarded the NVIDIA Graduate Fellowship.

My recent work includes designing bandwidth-centric servers using the emerging CXL memory interconnect. Previously, I worked on practical and secure defenses against the severe Rowhammer security threat.
